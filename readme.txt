Задание 1

Сверстать элемент согласно эскиза.

Требования:
- минимальный html
- цвет фона и цвет бордюра должны меняться через css, желательно -- несложными правилами, очень желательно -- по одному правилу на цвет
- Картинка вставяется через html или css
- Номер вставляется в html

Реализация:
	<div class="achievement achievement-no-number">
		<img src="owl.png">
	</div>

	<div class="achievement">
		<img src="owl.png">
		<span>2</span>
	</div>

	<div class="achievement achievement-gold">
		<img src="ddd.png">
		<span>99</span>
	</div>

css:
.achievement-gold {
	background-color: #ffff79;
	color: #fff016;
	fill: #fff016;
	/*Границу буду рисовать тенями. Тени по умолчанию наследуют цвет шрифта.*/
}

Примечание: изображение я не обрезаю по следующим причинам:
- если использовать .achievement {overflow: hidden}, то обрезается не только изображение
- если само изображение скруглять по углам, можно его испортить, если его размер меньше ачивки
Поэтому пусть об этом думает дизайнер. К тому же, я не связываю дизайнеру руки насчёт изображений с выступающими
хвостиками
